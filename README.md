# vue3-pomodoro

## Description

A practice project used to learn new frontend skills.

Tech Stack:
- Vue 3
- TypeScript
- Tailwind Css
- Vuex 4

UI design credits to 建良. [Prorotype](https://challenge.thef2e.com/user/3?schedule=2590#works-2590)
